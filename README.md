# circRNA Data Analysis

This repository contains the analysis pipelines and exploratory logs for circRNA data.

## Structure

The primary logs and associated analysis notebooks are stored in the `docs/` directory:

*   **`docs/01_on_data/data_journal.md`**: logs and information regarding the circRNA count matrices, junction types, and Differential Expression (DE) / Gene Set Enrichment Analysis (GSEA) stuff.
*   **`docs/01_on_data/python_env.md`**: instructions for setting up the `circrna` python environment.
